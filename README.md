# vue-shopping-cart
使用vue 2.0开发的购物车功能

## 如何预览
1. 下载仓库
```
git clone git@github.com:DanielXH/vue-shopping-cart.git
```

2. 使用node运行
```
node server.js
```

3.在浏览器中打开页面
```
http://localhost:3000/cart.html
```
